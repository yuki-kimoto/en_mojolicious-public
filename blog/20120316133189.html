SQLiteの情報を見るためのMojoliciousプラグイン Mojolicious::Plugin::SQLiteViewerLite

お知らせ

Mojolicious::Plugin::SQLiteViewerLiteはMojolicious::Plugin::DBViewerに統合されました。

Mojolicious::Plugin::SQLiteViewerLiteをCPANで公開しました。Mojoliciousのプラグインで、SQLiteのテーブル一覧やテーブルの定義をブラウザで表示させて観覧することができます。phpMyAdminの観覧専用のSQLiteのPerlバージョンだと思ってくれるとわかりやすいかもしれません。機能はずっとシンプルです。

先日公開した便利なMojolicious::Plugin::MySQLViewerLiteのSQLiteバージョンです。同じインターフェースで利用することができます。

  • Mojolicious::Plugin::SQLiteViewerLite

使い方

# Mojolicious::Lite
plugin 'SQLiteViewerLite', dbh => $dbh;

# Mojolicious
$app->plugin('SQLiteViewerLite', dbh => $dbh);

# Access
http://localhost:3000/SQLiteviewerlite

機能

Mojolicious::Plugin::SQLiteViewerLiteは次のような機能をもっています。

  • すべてのテーブル名の表示
  • show create tableの表示
  • select * from TABLE limit 0, 1000
  • すべてのテーブルのプライマリーキー、Nullが許可された列を表示
