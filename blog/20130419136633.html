Mojolicious::Plugin::DBViewer --Plugin for viewing database information in a browser

Mojolicious::Plugin::DBViewer strong> is a Mojolicious plugin for viewing database information from a browser .

Functions

  • Supports MySQL and SQLite
  • List table names
  • Displaying the contents of CREATE TABLE
  • Performing a search with a simple SELECT statement. Support for multiple search criteria, join
  • Display primary key, nullable columns, database engine, character encoding, etc.

Overview

Mojolicious::Plugin::DBViewer integrates Mojolicious::Plugin::MySQLViewerLite and Mojolicious::Plugin::SQLiteViewerLite to add functionality. The screen layout has also been cleaned up.

Please use Mojolicious::Plugin::DBViewer from now on. Note that you can no longer pass database handles directly. Be sure to specify the data source name for dsn.

It supports MySQL and SQLite, and you can see the database information from your browser. I think it's easy to imagine if you can think of it as something like phpMyAdmin for viewing only. It can be plugged in as a Mojolicious plugin.

How to use

Use it as follows. 

#Mojolicious::Lite
plugin (
  'DB Viewer',
  dsn => "dbi: mysql: database = bookshop",
  user =>'ken',
  password =>'! LFKD%$&'
);

#Mojolicious
$app->plugin(
  'DB Viewer',
  dsn => "dbi: mysql: database = bookshop",
  user =>'ken',
  password =>'! LFKD%$&'
);

You can access it at the following URL.

/ dbviewer
