Mojolicious Tutorial

Install Mojolicious

Mojolicious is just a Perl module, so it's as easy to install as any other module. 

cpan Mojolicious

If you want to try Mojolicious on Windows, this should be enough ( What to do if you get a YAML error. Also, WebSocket tests can be done depending on the timing. In some cases, it will succeed after several installations.) If you want to install Mojolicious in your environment on Linux or Mac OS X, you can refer to the following steps.

  • Building a local environment with cpanm (CentOS)

Also, Mojolicious does not depend on other modules, so you can upload it by FTP and use it easily on a rental server. You can refer to the following steps for this procedure.
